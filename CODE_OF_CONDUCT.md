# Code of Conduct

## Our pledge

This project adopts the [Contributor Covenant](https://www.contributor-covenant.org/),
version 2.1, as its code of conduct. The full text is published at:

https://www.contributor-covenant.org/version/2/1/code_of_conduct/

In short: everyone who participates here (contributors, maintainers, issue
reporters, reviewers) is expected to make participation a harassment-free
experience for others, to act in good faith, and to treat disagreement with
respect. Harassment, personal attacks, and discriminatory or demeaning
behavior are not welcome.

## Scope

This applies in all project spaces (the repository, issues, pull requests,
discussions) and whenever someone is representing the project in public.

## Reporting

If you experience or witness unacceptable behavior, report it through one of:

- GitHub's private vulnerability reporting form on this repository, marked as a
  conduct report, or
- a direct message to the repository owner via GitHub
  ([@mustafa-bharmal-shipveho](https://github.com/mustafa-bharmal-shipveho)).

Reports are handled confidentially. As this is currently a single-maintainer
project, there is no separate review board; if the project gains
co-maintainers, a dedicated reporting alias will replace the direct-contact
path above and be documented here.

## Enforcement

The maintainer may remove, edit, or reject contributions and comments that
violate this code, and may temporarily or permanently ban any contributor for
behavior they judge inappropriate, threatening, or harmful. Enforcement
decisions follow the impact-based guidelines described in the Contributor
Covenant v2.1 (correction, warning, temporary ban, permanent ban), linked
above.
